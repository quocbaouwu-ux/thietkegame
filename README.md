# Game Đoán Số C++

## Giới thiệu
Game đoán số đơn giản viết bằng C++. Người chơi đoán số từ 1 đến 100.

## Chức năng
- Thêm người chơi
- Xem danh sách người chơi
- Chơi game đoán số
- Lưu điểm vào file

## Cấu trúc
main.cpp  : Source code chương trình
players.txt : Dữ liệu người chơi
README.md : Mô tả project

## Cách chạy
1. Mở main.cpp bằng IDE C++
2. Compile
3. Run chương trình
